## getKeypair

```curl
curl -X GET \
  https://testnet-horizon.triamnetwork.com/getKeypair \
  -H 'cache-control: no-cache' \
  -H 'content-type: application/json' \
  -H 'postman-token: 87887390-8ad8-1e95-7046-5dbf6076cf89'
```

